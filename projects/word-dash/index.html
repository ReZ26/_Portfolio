Unreal Engine Project

Word Dash

Overview: An Unreal Engine 5.5 multiplayer word game built mostly in Blueprints. It combines character-based gameplay with a custom word-game loop, session flow, and competitive player interaction, all managed through a full UI flow from main menu to end-game screens.

Download Build

Technical Breakdown

Game Framework Architecture

  • GameInstance for persistent cross-level state and global game flow management
  • GameMode handling match rules, player spawning, and session initialization
  • GameState for shared match data replicated to all clients in real-time
  • PlayerController managing input, camera, and player-specific UI updates
  • PlayerState tracking individual player score, stats, and gameplay progression

Multiplayer & Session System

  • Session-based multiplayer flow with dedicated menu and lobby maps
  • Travel map for seamless transitions between menu and gameplay levels
  • Player list and server browser for session discovery and joining
  • Replicated player state for synchronized gameplay across all clients
  • Server-authoritative word validation and score calculation

Word Game System

  • Data-driven word list loaded from configuration files
  • Word-formation logic with valid word validation against dictionary
  • Drop-letter system allowing players to discard and redraw letters
  • Score calculation based on word length, letter values, and multipliers
  • Turn-based gameplay flow with timed actions and player rotation

UI & Menu Flow

  • Main menu with session creation and joining options
  • Lobby system showing connected players and game readiness
  • In-game HUD displaying current word, score, and player turn indicators
  • Pause menu with settings and return-to-menu options
  • Loading screen with progress tracking during map transitions
  • End-game screen displaying final scores and leaderboard rankings

Progression & Data

  • Save-game system for player progress and statistics persistence
  • Leaderboard tracking with accuracy metrics and win/loss records
  • Player profile data including gameplay history and achievements
  • ScriptableObject-driven configuration for game parameters and balance
  • Audio system with sound effects and UI feedback cues

Blueprint Implementation

  • Entirely Blueprint-driven gameplay logic for rapid iteration
  • Reusable blueprint components for UI widgets and game mechanics
  • Event-driven communication between systems and UI updates
  • Animation blueprints for character and UI state transitions
  • Material instances for consistent visual styling across the interface

Description

  • Built in Unreal Engine 5.5 as a multiplayer word game using a Blueprint-first workflow.
  • Combines character-based gameplay with a custom word-game loop where players form or drop words.
  • Includes dedicated maps for main menu, lobby, travel flow, and gameplay levels.
  • Uses GameInstance for global flow, with GameMode and GameState handling match rules and shared session state.
  • Uses PlayerController and PlayerState for player-specific flow, interaction, and score tracking.
  • Includes a data-driven word list, save-game support, and full UI coverage with menu and HUD widgets.
  • Adds sound effects and interface polish to improve clarity and player experience.
