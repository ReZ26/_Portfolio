Slimey Jump

  • Cling logic for wall/platform attachment and movement variants

Description

Slimey Jump is a 2D vertical endless platformer that combines arcade action with deep progression systems. Players navigate upward through procedurally generated platforms using drag-and-release jump mechanics with trajectory preview, while managing dynamic camera pressure that increases difficulty. The game features a complete progression ecosystem including coin-based economy, card-based permanent buffs, daily/weekly quests, shop customization, live leaderboards with online submission, and a hybrid idle mode for offline reward generation. The architecture supports multiple gameplay layers: moment-to-moment platformer action, meta progression through cards and quests, and long-term retention through idle mechanics and cosmetic customization.
