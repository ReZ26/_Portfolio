Unity Engine Project

Slimey Jump

Overview: A charming platformer built in Unity featuring a slime protagonist with unique physics-based movement. The game combines smooth jump mechanics with slime-specific interactions, creating engaging level design and challenging progression.

Play Game GitHub

Images

Description

Slimey Jump is a carefully crafted platformer that leverages slime physics to create unique gameplay opportunities. Players control a gelatinous protagonist that can stretch, bounce, and adapt to environmental challenges. The game features carefully tuned jump mechanics, responsive controls, and clever level design that teaches players the slime's unique abilities.

What I Have Done

  • Developed slime-based physics and movement systems with Unity's 2D Rigidbody2D
  • Created responsive jump mechanics with variable jump heights and timing
  • Designed level progression with escalating difficulty and new mechanics introduction
  • Implemented collision detection and interaction systems for environmental puzzles
  • Built UI systems for level selection, score tracking, and pause menus
  • Created particle systems and visual feedback for slime interactions
