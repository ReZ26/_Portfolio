Description

A 3D action-adventure prototype developed in Unreal Engine 5.4, built on the Third Person template and expanded with custom combat, boss encounters, UI flow, animation, and environmental effects. The game includes multiple gameplay scenes, a dedicated boss sequence, player HUD systems, pause/settings menus, and end-game screens.

What I have done in this project

  • Built the core gameplay structure using the Unreal Engine Third Person framework.
  • Created multiple gameplay maps and level transitions, including a dedicated boss fight scene.
  • Implemented a boss encounter flow with custom interaction scenes.
  • Set up a projectile-based attack system with elemental variants (Dark Energy, Fire, Ice).
  • Developed a full UI/UX flow, including a main menu, in-game HUD, boss health bars, pause/settings menus, and end-game screens.
  • Integrated character animations and montages for combat and movement.
  • Added teleportation mechanics and custom player/enemy folder organization.
  • Utilized Unreal Engine's Water and PCG plugins for environmental effects.
  • Organized project content into modular folders to keep gameplay systems maintainable.
