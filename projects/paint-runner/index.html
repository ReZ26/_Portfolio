Unity Engine Project

Paint Runner

Overview: A vibrant 2D action game built in Unity where players unleash dynamic painting mechanics combined with engaging progression systems. The game features vibrant visual feedback and challenging gameplay that keeps players engaged throughout.

Description

Paint Runner is a dynamic 2D action game that combines intuitive movement mechanics with innovative painting-based gameplay. Players navigate through vibrant levels leaving trails of color while overcoming obstacles and enemies. The game emphasizes responsive controls, visual feedback, and satisfying progression systems that reward player skill and creativity.

What I Have Done

  • Built core 2D movement and painting mechanics using Unity's physics system
  • Implemented dynamic visual feedback systems with particle effects and color systems
  • Created progression and difficulty scaling systems for engaging gameplay
  • Designed UI/UX for menus, HUD, and player feedback elements
  • Optimized performance for smooth 60 FPS gameplay on mobile and desktop
  • Integrated audio systems for sound effects and music feedback
