Description

It's Me Pico is a competitive territory-control mobile game where the main goal is to control more tiles than an AI opponent before the match ends. The game starts with an arena of neutral paintable tiles—as the player or AI moves over tiles, ownership changes in real time and the score updates dynamically. Beyond the core loop, the game includes daily and weekly quest systems for engagement, a daily reward streak system for retention, multiple power-up types that alter match outcomes, level progression with AI difficulty scaling, and complete monetization systems including rewarded ads, currency economy, lives, and shop. This project demonstrates gameplay programming and production-level system integration suitable for a shipped mobile game.
