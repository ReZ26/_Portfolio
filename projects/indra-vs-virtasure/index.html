Description

Legend of Indra vs Vritrasur is a long-form action-adventure RPG built in Unreal Engine 5, inspired by the ancient Hindu epic of the war between Indra, the king of gods, and Vritrasur, the cosmic dragon demon. I served as both Game Designer and Developer on this project — responsible for the combat system, enemy AI, narrative flow, and environment design. The game was built and iterated across multiple months and showcased at the CU Project Expo 2025, where it won both 1st and 3rd Prize. The project features 3D immersive environments with atmospheric Unreal Engine 5 lighting, Blueprint-driven gameplay systems, and a deeply balanced progression curve crafted through extensive iterative playtesting.

What I have done in this project

  • Developed cutscenes based on the given direction, ensuring smooth narrative flow.
  • Designed and implemented a quest system to manage player objectives and progression.
  • Created a narration system to enhance storytelling and player immersion.
  • Built an inventory system for efficient item management and player interaction.
