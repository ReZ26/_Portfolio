Description

This project is a Blueprint-based Unreal Engine 5.4 third-person game prototype focused on a time-rewind gameplay mechanic. The main idea is to let the player record movement and actions in real time, then reverse and replay them with clear visual and audio feedback. Alongside the rewind feature, the project includes core third-person movement, combat interactions, UI hints, and effect systems to make the mechanic feel responsive and polished.

What I have done in this project

  • Built the core rewind system to capture and restore player state over time.
  • Structured rewind data and ghost/trail data so past player actions can be replayed clearly.
  • Integrated rewind visuals using dedicated materials and trail effects for better gameplay readability.
  • Added rewind audio cues and transition feedback to improve player feel.
  • Connected the system to the third-person character and game mode flow.
  • Set up and used Enhanced Input mappings for gameplay, inventory, and crafting contexts.
  • Added supporting UI hint behavior to communicate actions to the player.
  • Organized project assets and level setup in the main map for testing and iteration.
  • Developed the prototype fully in Blueprints without C++ to keep iteration fast.
