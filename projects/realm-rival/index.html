Description

An Unreal Engine 5.5 Blueprint-driven multiplayer shooter prototype focused on core gameplay systems, character abilities, objective flow, and menu-to-match transitions. It is designed as a scalable multiplayer foundation where players can spawn into matches, use combat abilities, and interact with objective-based gameplay like capturing flags or securing areas.

What I have done in this project

  • Core Multiplayer Structure: Set up a dedicated game framework with custom GameModes and a travel map for menu-to-match flow.
  • Menu and Map Flow: Configured the project to start in a main menu map and support a lobby-based pre-game flow.
  • Gameplay Systems: Added Blueprint components for character abilities, a damage system, status effects, and objective actors (flags, secure areas).
  • Character and Content Integration: Integrated custom character assets and organized content by feature area (Weapons, Projectiles, Effects, etc.).
  • Input and Controls: Enabled and configured the Enhanced Input system for keyboard, mouse, and gamepad.
  • Visual and Platform Setup: Configured UE5 rendering features like Lumen and Nanite and set up packaging configurations for builds.
  • Project Organization: Structured content into clear domains and added plugin support to keep the project extensible.
