Description

A third-person Unreal Engine character project built around the Advanced Locomotion System (ALS), extended with weapon handling, animation layers, interaction systems, damage logic, and UI elements. This project focuses on building a responsive, game-ready character framework using ALS as the movement foundation, supporting multiple combat and traversal behaviors.

What I have done in this project

  • Set up and organized the ALS character project structure in Unreal Engine.
  • Integrated rifle and pistol mechanics with dedicated animation blueprints and equip content.
  • Added and organized locomotion, aiming, and transition animation assets.
  • Prepared gameplay interfaces for interaction and damage handling.
  • Built the foundation for a modular damage system using enums and structs.
  • Added UI/widget-related content for gameplay feedback and character presentation.
  • Organized VFX/material/particle assets to support combat and movement feedback.
  • Structured content folders for easier iteration and future feature expansion.
