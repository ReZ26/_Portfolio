Technical Breakdown

Damage System

  • Modular damage framework with BPC_DamageSystem component
  • Damageable interface for consistent implementation
  • Damage type enums with response handling
  • Health bar UI widget integration

Combat Architecture

  • Attack data component (BPC_Attacks) with S_AttackInfo struct
  • Melee and ranged combat support
  • Animation-based hit detection
  • Cooldown and resource management

Enemy System

  • Enemy base Blueprint with shared behavior
  • Specialized variants (melee, ranged, werewolf/boss)
  • AI decision-making and target evaluation
  • Per-archetype animations and VFX

Ability Framework

  • Area-of-effect (AOE) base Blueprint system
  • Instant and over-time effect variants
  • Effect types (blast, lightning, fire, etc.)
  • Configurable radius and duration parameters

Input & Controls

  • Enhanced Input System for modern input handling
  • Movement, aiming, sprint, crouch, jump actions
  • Interact and traversal key mappings
  • Controller and keyboard/mouse support

World & Content

  • Multiple themed environments (sci-fi, fantasy, planets)
  • Modular plugin architecture (DamageSystem, Interaction)
  • Reusable VFX/material libraries
  • Organized content pipeline by feature domain

Description

An Unreal Engine action-combat game prototype built around a third-person gameplay loop in a sci-fi/fantasy world. The project combines character combat, enemy AI, VFX-driven abilities, and exploration-focused level design, supporting melee/ranged combat, multiple enemy archetypes, and various themed environments.

What I have done in this project

  • Core Gameplay Systems: Set up a modular damage framework, combat foundations with attack data, and an AOE ability system.
  • Character and Enemy Setup: Prepared a Blueprint hierarchy for enemies (base, melee, ranged, boss variants) and integrated character/animation resources for combat.
  • Input and Control Layer: Configured Enhanced Input for movement, combat, and interaction, with support for controller input.
  • World Building and Content: Organized multiple content packs and map spaces (sci-fi, fantasy, etc.) and integrated reusable VFX/material libraries.
  • UI and Feedback: Added health bars and damage response assets to provide clear combat feedback.
