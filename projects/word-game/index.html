Unity | Location-Based Word Puzzle

Word Game

Overview: Location-themed word puzzle game where players travel across destinations, solve progressively difficult word levels, and manage economy-driven meta progression with room systems, NPCs, and collectible rewards.

Technical Breakdown

Core Game Systems

  • GameController managing session values, level progression, and state tracking
  • LevelControllerBase and NormalLevelController building and running puzzle levels
  • WordPickerSO data-driven word selection with difficulty and target logic
  • Progressive level generation with end-of-level reward and UI triggers

Progression & Navigation

  • MapButton location selection with lock/unlock flow and persistent state
  • PlayLocation animated transition with delayed gameplay start
  • Location unlock progression with next-location logic
  • UINavigator managing room and game transitions seamlessly

Economy & Market Systems

  • ShopDisplay singleton UI controller for market visibility
  • Currency management and AddButtonCurrency routing system
  • Chest system with slot management and full-slot feedback
  • ScriptableObject-driven economy balancing without hardcoding

Tutorial & Onboarding

  • Tutorial and TutorialCockpitRoom with gating thresholds
  • Event-driven tutorial behavior with milestone progression
  • Guided hand-sequence interactions in cockpit tutorial
  • ClickableAreaHighlighter contextual UI with dynamic updates

NPC & Interaction

  • RandomWalkNPC with timed dialogue trigger behavior
  • Smooth NPC movement without boundary jitter
  • Event-driven NPC interaction system
  • Character animations and dialogue integration

Presentation & Polish

  • High-quality sprite package for enhanced visual fidelity
  • DOTween-powered animated transitions
  • Location label fade transitions for smooth UX
  • Animation controllers for room and UI presentation

Description

Zerzura Travel Word Game is a Unity-based location-themed word puzzle experience where players travel across destinations solving progressively difficult word levels. The game combines core word puzzle mechanics with meta progression through economy systems, room interactions, NPC dialogue, and collectible rewards. Players progress through location unlocks, manage resources via market and chest systems, and complete room activities while building character progression.

What I Have Done

  • Architected modular game systems with GameController managing session values and progression state
  • Built level generation system using LevelControllerBase and WordPickerSO with data-driven difficulty scaling
  • Implemented location progression with MapButton lock/unlock flow and persistent state management
  • Designed animated location transitions in PlayLocation with delayed gameplay start for smooth UX
  • Created economy systems with ShopDisplay, currency management, and ScriptableObject-driven balancing
  • Developed tutorial system with gating thresholds, guided interactions, and milestone progression triggers
  • Built NPC interaction system with RandomWalkNPC timed dialogue and smooth movement behavior
  • Implemented chest system with slot management and full-slot feedback UI
  • Integrated high-quality sprite assets and DOTween animations for polished presentation
  • Managed 712 file changes across scripts, prefabs, scenes, and assets for comprehensive feature update
