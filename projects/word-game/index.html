Unity Engine Project

Word Game

Overview: A competitive word-based game built in Unity that combines puzzle mechanics with engaging UI/UX. Players compete in fast-paced word challenges while enjoying a polished interface and rewarding progression system.

Download Build GitHub

Images

Description

Word Game is a competitive gaming experience that challenges players to think quickly and strategically about word construction and placement. The game features real-time competition, word validation systems, and a clean, intuitive UI that makes gameplay accessible to all skill levels. Players can compete against AI or other players in various game modes.

What I Have Done

  • Developed word validation and scoring systems using dictionary databases
  • Implemented game state management for single-player and multiplayer modes
  • Created polished UI systems using Unity UI and TextMeshPro
  • Built leaderboard and progression tracking systems
  • Designed responsive layouts for various screen sizes
  • Implemented turn-based and real-time game modes
