Unity Engine Project

Surprise Sprint

Overview: A dynamic runner game built in Unity that keeps players on their toes with unpredictable obstacles and exciting power-up systems. The game features responsive character controls, escalating difficulty, and satisfying progression mechanics.

Download Build GitHub

Images

Description

Surprise Sprint is an adrenaline-pumping runner game that emphasizes unpredictability and player reaction. With randomly generated obstacles, powerful power-ups, and dynamic difficulty scaling, each run feels fresh and challenging. The game rewards skill, quick reflexes, and strategic power-up usage while maintaining a fun, accessible gameplay experience.

What I Have Done

  • Implemented responsive character controller with smooth movement and jumping
  • Created procedural obstacle generation system for varied gameplay
  • Developed power-up systems with visual and audio feedback
  • Built difficulty scaling and progression systems
  • Implemented collision detection and hit feedback systems
  • Created UI for score tracking, high scores, and game over screens
