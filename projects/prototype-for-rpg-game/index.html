Description

A third-person action RPG/combat prototype built in Unreal Engine. The project combines C++ core character logic with a large Blueprint-driven gameplay layer that includes melee and ranged enemies, attack systems, damage handling, interactive traps/objects, UI widgets, and area-of-effect abilities.

The game foundation follows a modular structure so gameplay features can be expanded quickly through components, interfaces, enums, structs, and reusable base Blueprints.

What I have done in this project

  • Core Player Controller: Set up a third-person character with spring arm camera, enhanced input, and combat state checks.
  • Combat & Attack Framework: Built an attack architecture with data-driven attack info and animation-notify based collision tracing.
  • Damage System: Implemented a modular damage system with a `Damageable` interface, damage types, and health bar support for reuse across actors.
  • Enemy & AI Content: Created a base enemy class and specialized variants (boss, mage, melee, ranged) with unique animations and VFX.
  • Area-of-Effect (AOE) System: Added a reusable AOE Blueprint base for instant and over-time effects like explosions, fire, and healing.
  • Interactable & Trap Systems: Implemented a base framework for interactable objects and traps using an interaction interface.
  • Modular Design: Used component-based design and interfaces to keep systems decoupled and extensible.
  • UI & Feedback: Added gameplay widgets for health bars and enemy UI, plus trail and damage visual effects.
  • Content Organization: Structured all assets by system (Animation, Enemy, DamageSystem, etc.) to maintain a clean and scalable project.
