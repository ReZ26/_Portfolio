Unity Engine Project

Bubble Trap

Overview: An engaging puzzle-action game built in Unity that combines strategic thinking with fast-paced gameplay. Players manipulate bubbles to solve puzzles and progress through increasingly complex challenges while enjoying satisfying visual effects.

Download Build GitHub

Images

Description

Bubble Trap is a innovative puzzle-action game that challenges players to think strategically about bubble placement and interactions. The game features bubble physics, chain reactions, and satisfying visual feedback that creates an addictive gameplay loop. Each level presents new mechanics and challenges that build upon the core bubble manipulation system.

What I Have Done

  • Implemented bubble physics and collision detection systems
  • Created chain reaction mechanics with visual feedback
  • Designed progressive level systems with escalating complexity
  • Built puzzle interaction and solution validation systems
  • Developed particle effects for bubble popping and interactions
  • Created UI systems for level progression and scoring
